# microblog-frontEnd

本项目是[microblog项目的前端项目]()，使用vue＋element-ui进行开发．

## 环境配置

```bash
#安装vue基础模块
npm install

#安装element-ui
npm install element-ui -S

#安装axios
npm install axios

#安装vue-router
npm install vue-router
```

## 运行打包

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
